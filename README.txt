QT Cinema Manager Project

Resource files/: Resources such as images and UI files.
x64/Debug/: The directory where the database file should be placed during development.

Setup Instructions
Clone the repository to your local machine.

Ensure you have the necessary dependencies installed:

Qt framework
Visual Studio 2022
Open the project in your VS22

Database Setup:

Move the database file to the x64/Debug/ folder within the project directory. This step is crucial for the application to connect to the database correctly during development.
